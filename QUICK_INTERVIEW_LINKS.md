### 30-Minute Review Checklist
- [ ] Review Python basics: [Python Cheat Sheet](https://www.pythoncheatsheet.org/)
- [ ] SQL joins and window functions: [SQL Cheat Sheet](https://www.sqltutorial.org/sql-cheat-sheet/)
- [ ] Cloud services overview: [AWS/Azure/GCP Comparison](https://comparecloud.in/)
- [ ] Data pipeline concepts: [Data Engineering Concepts](https://github.com/andkret/Cookbook)
- [ ] Practice 2-3 easy coding problems: [LeetCode Easy](https://leetcode.com/problemset/all/?difficulty=Easy)

### Common Questions to Review
1. "Explain your data pipeline architecture"
2. "How do you handle data quality issues?"
3. "Difference between batch and stream processing"
4. "How do you optimize SQL queries?"
5. "Explain CAP theorem"
6. "How do you handle schema evolution?"
7. "Describe your experience with cloud platforms"
