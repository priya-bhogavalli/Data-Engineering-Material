# Data Structures & Algorithms Resources

## Essential Books

### Foundational Texts
1. **"Introduction to Algorithms" (CLRS)** by Cormen, Leiserson, Rivest, Stein
   - Comprehensive coverage of algorithms and data structures
   - Mathematical rigor with detailed proofs
   - Best for: Deep understanding and academic reference

2. **"Algorithms" by Robert Sedgewick & Kevin Wayne**
   - Practical approach with Java implementations
   - Excellent visualizations and examples
   - Best for: Implementation-focused learning

3. **"The Algorithm Design Manual" by Steven Skiena**
   - Problem-solving approach with war stories
   - Practical advice for algorithm selection
   - Best for: Real-world application guidance

### Interview Preparation
4. **"Cracking the Coding Interview" by Gayle McDowell**
   - 189 programming questions with solutions
   - Interview strategies and tips
   - Best for: Technical interview preparation

5. **"Elements of Programming Interviews" by Aziz, Lee, Prakash**
   - Language-specific versions (Python, Java, C++)
   - Detailed solutions with complexity analysis
   - Best for: In-depth interview preparation

6. **"Programming Pearls" by Jon Bentley**
   - Classic problem-solving techniques
   - Elegant solutions to complex problems
   - Best for: Developing algorithmic thinking
