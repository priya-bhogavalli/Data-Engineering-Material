# Trifacta (Google Cloud Dataprep) - Key Concepts

## Overview
Trifacta is a data preparation platform that enables users to clean, structure, and enrich raw data for analysis. It's now part of Google Cloud as Cloud Dataprep by Trifacta.

## Core Features
- **Visual data preparation**: Interactive, visual interface for data wrangling
- **Machine learning suggestions**: AI-powered data transformation recommendations
- **Self-service**: Business users can prepare data without coding
- **Scalable processing**: Runs on Apache Spark and Google Cloud Dataflow
- **Data profiling**: Automatic data quality assessment and anomaly detection

## Key Capabilities
- **Data discovery**: Automatic data profiling and pattern detection
- **Data transformation**: 300+ built-in transformation functions
- **Data quality**: Automated data quality scoring and validation
- **Collaboration**: Team-based data preparation workflows
- **Integration**: Connect to 100+ data sources and destinations

## Supported Data Sources
- **Cloud storage**: Google Cloud Storage, Amazon S3, Azure Blob
- **Databases**: BigQuery, Snowflake, Redshift, PostgreSQL, MySQL
- **Files**: CSV, JSON, Parquet, Avro, Excel
- **Streaming**: Kafka, Pub/Sub
- **Applications**: Salesforce, ServiceNow, SAP

## Transformation Engine
- **Recipe-based**: Step-by-step transformation recipes
- **Visual feedback**: Real-time preview of transformations
- **Undo/redo**: Full transformation history
- **Parameterization**: Reusable transformation templates
- **Scheduling**: Automated data preparation workflows
