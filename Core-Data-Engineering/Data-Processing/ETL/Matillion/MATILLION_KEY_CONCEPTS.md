# Matillion - Key Concepts

## Overview
Matillion is a cloud-native data integration platform designed specifically for cloud data warehouses like Snowflake, Amazon Redshift, Google BigQuery, and Azure Synapse.

## Core Architecture
- **Cloud-native**: Built specifically for cloud data warehouses
- **ELT approach**: Extract, Load, Transform using warehouse compute
- **Visual interface**: Drag-and-drop job design
- **Orchestration**: Workflow orchestration and scheduling
- **Transformation**: In-warehouse data transformation using SQL

## Key Components
- **Matillion ETL**: Visual data integration and transformation
- **Matillion Data Loader**: High-speed data loading
- **Orchestration Jobs**: Workflow management and scheduling
- **Transformation Jobs**: Data transformation using warehouse compute
- **API Hub**: Pre-built connectors and integrations

## Supported Platforms
- **Snowflake**: Native Snowflake integration
- **Amazon Redshift**: AWS data warehouse integration
- **Google BigQuery**: GCP data warehouse integration
- **Azure Synapse**: Microsoft data warehouse integration
- **Databricks**: Unified analytics platform integration

## Data Sources
- **Databases**: MySQL, PostgreSQL, Oracle, SQL Server, MongoDB
- **SaaS Applications**: Salesforce, HubSpot, Zendesk, ServiceNow
- **Cloud Storage**: S3, GCS, Azure Blob, SFTP
- **Files**: CSV, JSON, Parquet, Avro, XML
- **APIs**: REST APIs, webhooks, custom connectors

## Transformation Engine
- **SQL-based**: Leverage warehouse SQL capabilities
- **Component library**: 100+ pre-built transformation components
- **Custom SQL**: Write custom SQL transformations
- **Python integration**: Execute Python scripts
- **Machine learning**: ML model integration and scoring
