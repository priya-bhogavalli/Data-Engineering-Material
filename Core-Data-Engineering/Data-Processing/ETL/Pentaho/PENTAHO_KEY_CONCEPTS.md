# Pentaho - Key Concepts

## Overview
Pentaho is a comprehensive business intelligence and data integration platform that provides ETL, data mining, OLAP, reporting, and dashboard capabilities.

## Core Components
- **Pentaho Data Integration (PDI)**: ETL tool (formerly Kettle)
- **Pentaho Business Analytics**: BI and reporting platform
- **Pentaho Data Mining**: Advanced analytics and machine learning
- **Pentaho Reporting**: Report generation and distribution
- **Pentaho Dashboards**: Interactive dashboard creation

## Key Features
- **Visual ETL**: Drag-and-drop ETL job design
- **Big data integration**: Hadoop, Spark, NoSQL support
- **Real-time processing**: Stream processing capabilities
- **Data quality**: Built-in data profiling and cleansing
- **Metadata management**: Centralized metadata repository
- **Enterprise features**: Scheduling, monitoring, security

## ETL Capabilities
- **Transformations**: Data transformation workflows
- **Jobs**: Orchestration and control flow
- **Steps**: Individual transformation operations
- **Hops**: Data flow connections between steps
- **Variables**: Dynamic parameterization
- **Error handling**: Comprehensive error management

## Data Sources
- **Databases**: All major relational databases
- **Big data**: Hadoop, Spark, Cassandra, MongoDB
- **Cloud**: AWS, Azure, Google Cloud services
- **Files**: CSV, Excel, XML, JSON, fixed-width
- **Applications**: SAP, Salesforce, web services
- **Streaming**: Kafka, JMS message queues
