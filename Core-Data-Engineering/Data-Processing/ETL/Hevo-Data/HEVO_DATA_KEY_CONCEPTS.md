# Hevo Data - Key Concepts

## Overview
Hevo Data is a no-code data pipeline platform that provides automated data integration from various sources to data warehouses and databases with real-time sync capabilities.

## Core Features
- **No-code platform**: Visual pipeline builder
- **Real-time sync**: Sub-15 minute data replication
- **150+ integrations**: Pre-built source connectors
- **Auto-schema mapping**: Automatic schema detection and mapping
- **Data transformation**: Built-in transformation capabilities
- **Monitoring**: Real-time pipeline monitoring and alerting

## Supported Sources
- **Databases**: MySQL, PostgreSQL, MongoDB, Oracle, SQL Server
- **SaaS Applications**: Salesforce, HubSpot, Google Analytics, Facebook Ads
- **Cloud Storage**: Amazon S3, Google Cloud Storage, Azure Blob
- **Files**: CSV, JSON, XML, Parquet formats
- **APIs**: REST APIs and webhooks
- **Streaming**: Apache Kafka, Amazon Kinesis

## Supported Destinations
- **Cloud Warehouses**: Snowflake, Amazon Redshift, Google BigQuery
- **Databases**: PostgreSQL, MySQL, SQL Server
- **Data Lakes**: Amazon S3, Google Cloud Storage
- **Analytics**: Tableau, Looker, Power BI integration

## Key Capabilities
- **Change Data Capture**: Real-time database replication
- **Schema evolution**: Automatic schema change handling
- **Data quality**: Built-in data validation and cleansing
- **Error handling**: Robust error recovery mechanisms
- **Security**: End-to-end encryption and compliance
