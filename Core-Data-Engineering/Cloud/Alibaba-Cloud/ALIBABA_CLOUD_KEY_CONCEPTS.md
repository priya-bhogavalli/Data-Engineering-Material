# Alibaba Cloud - Key Concepts

## Overview
Alibaba Cloud is a leading cloud computing platform that provides comprehensive cloud services including computing, storage, networking, security, and big data analytics. It's particularly strong in the Asia-Pacific region and offers innovative solutions for digital transformation.

## Core Computing Services

### Elastic Compute Service (ECS)
- **Virtual Machines**: Scalable virtual server instances
- **Instance Types**: Various configurations for different workloads
- **Auto Scaling**: Automatic scaling based on demand
- **Spot Instances**: Cost-effective preemptible instances
- **Dedicated Hosts**: Physical servers for compliance requirements

### Container Services
- **Container Service for Kubernetes (ACK)**: Managed Kubernetes clusters
- **Elastic Container Instance (ECI)**: Serverless containers
- **Container Registry**: Private container image registry
- **Service Mesh**: Istio-based service mesh solution
- **Serverless Kubernetes**: Fully managed serverless Kubernetes

### Serverless Computing
- **Function Compute**: Event-driven serverless functions
- **Serverless App Engine**: Application hosting platform
- **Event Bridge**: Event routing and processing
- **Workflow**: Visual workflow orchestration
- **API Gateway**: Managed API gateway service

## Storage Services

### Object Storage Service (OSS)
- **Scalable Storage**: Unlimited object storage capacity
- **Multiple Storage Classes**: Standard, infrequent access, archive
- **Global Distribution**: Multi-region data replication
- **Data Processing**: Built-in image and video processing

### Database Services
- **ApsaraDB RDS**: Managed relational databases
- **PolarDB**: Cloud-native distributed database
- **AnalyticDB**: Real-time data warehouse
- **MongoDB**: Managed MongoDB service
- **Redis**: In-memory database service
