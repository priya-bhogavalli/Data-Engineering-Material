# Qlik Sense - Key Concepts

## Overview
Qlik Sense is a modern analytics platform that provides self-service data visualization and discovery capabilities. It features an associative analytics engine that enables users to explore data freely and uncover hidden insights through interactive visualizations.

## Core Architecture

### Associative Engine
- **Associative Model**: Unique data model that maintains relationships between all data elements
- **In-Memory Processing**: High-performance in-memory analytics engine
- **Columnar Storage**: Optimized columnar data storage for analytics
- **Compression**: Advanced compression algorithms for memory efficiency

### Multi-Cloud Platform
- **Cloud-Native**: Built for cloud deployment and scalability
- **Hybrid Deployment**: Support for on-premises, cloud, and hybrid deployments
- **Multi-Tenant**: Secure multi-tenant architecture
- **Elastic Scaling**: Automatic scaling based on demand
