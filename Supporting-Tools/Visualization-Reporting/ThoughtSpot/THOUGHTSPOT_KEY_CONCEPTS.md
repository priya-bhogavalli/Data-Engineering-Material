# ThoughtSpot - Key Concepts

## Overview
ThoughtSpot is a search-driven analytics platform that enables users to search and analyze data using natural language queries. It provides instant answers to business questions through an intuitive search interface, making analytics accessible to all users.

## Core Architecture

### Search-Driven Analytics
- **Natural Language Processing**: Understand and process natural language queries
- **Search Interface**: Google-like search experience for data
- **Instant Results**: Sub-second query response times
- **Auto-Suggestions**: Intelligent query suggestions and completions

### In-Memory Engine
- **Columnar Storage**: Optimized columnar data storage format
- **In-Memory Processing**: High-performance in-memory analytics engine
- **Compression**: Advanced compression algorithms for storage efficiency
- **Parallel Processing**: Massively parallel query execution

### Search Experience
