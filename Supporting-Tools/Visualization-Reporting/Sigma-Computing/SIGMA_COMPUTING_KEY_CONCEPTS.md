# Sigma Computing - Key Concepts

## Overview
Sigma Computing is a cloud-native business intelligence platform that enables users to explore, analyze, and visualize data using familiar spreadsheet-like interfaces. It provides real-time analytics capabilities with direct connections to cloud data warehouses.

## Core Architecture

### Cloud-Native Design
- **Browser-Based**: Fully web-based application with no desktop installation
- **Elastic Scaling**: Automatically scales compute resources based on demand
- **Multi-Tenant**: Secure multi-tenant architecture for enterprise deployment
- **API-First**: Comprehensive REST APIs for integration and automation

### Data Connection
- **Direct Warehouse Connection**: Connects directly to cloud data warehouses
- **Live Data**: Real-time queries without data movement or caching
- **Multiple Sources**: Support for Snowflake, BigQuery, Redshift, Databricks
- **Secure Connectivity**: Enterprise-grade security and encryption

## Key Features

### Spreadsheet Interface
- **Familiar UX**: Excel-like interface for business users
- **Drag-and-Drop**: Intuitive data manipulation and analysis
- **Formulas**: Support for complex calculations and functions
- **Pivot Tables**: Dynamic pivot table creation and manipulation

### Real-Time Analytics
- **Live Queries**: Direct queries to source systems
- **No ETL Required**: Eliminate data movement and transformation
- **Instant Updates**: Real-time data refresh and updates
- **Collaborative Analysis**: Multiple users working on same dataset

### Visualization Capabilities
- **Chart Types**: Comprehensive set of visualization options
- **Interactive Dashboards**: Dynamic and interactive dashboard creation
- **Custom Visualizations**: Extensible visualization framework
- **Mobile Responsive**: Optimized for mobile and tablet viewing
