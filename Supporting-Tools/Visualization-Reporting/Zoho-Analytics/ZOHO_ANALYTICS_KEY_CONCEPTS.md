# Zoho Analytics - Key Concepts

## Overview
Zoho Analytics is a self-service business intelligence and data analytics software that helps users create visually appealing data visualizations and insightful dashboards. It's part of the Zoho suite of business applications and offers both cloud-based and on-premises deployment options.

## Core Platform Features

### Self-Service Analytics
- **Drag-and-Drop Interface**: Intuitive report and dashboard creation
- **No-Code Analytics**: Create analytics without programming knowledge
- **Visual Query Builder**: Build queries using visual interface
- **Smart Recommendations**: AI-powered chart and insight suggestions
- **Natural Language Queries**: Ask questions in plain English

### Data Integration
- **250+ Integrations**: Pre-built connectors for various data sources
- **Cloud Applications**: Salesforce, Google Analytics, Facebook, Twitter
- **Databases**: MySQL, PostgreSQL, SQL Server, Oracle
- **File Imports**: Excel, CSV, JSON, XML file support
- **API Connections**: REST API and webhook integrations
- **Real-Time Sync**: Live data synchronization capabilities

## Data Preparation and Modeling

### Data Preparation
- **Data Cleansing**: Built-in data cleaning and transformation tools
- **Data Blending**: Combine data from multiple sources

### Data Modeling
