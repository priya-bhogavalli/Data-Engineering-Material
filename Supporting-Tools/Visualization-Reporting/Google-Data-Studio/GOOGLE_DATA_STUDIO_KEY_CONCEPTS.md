# Google Data Studio (Looker Studio) - Key Concepts

## Overview
Google Data Studio, now rebranded as Looker Studio, is a free web-based business intelligence tool that transforms data into informative, easy-to-read, customizable dashboards and reports. It integrates seamlessly with Google's ecosystem and various third-party data sources.

## Core Features

### Data Connectivity
- **Google Services**: Native integration with Google Analytics, Ads, Sheets, BigQuery
- **Third-Party Connectors**: 800+ partner connectors for various data sources
- **Database Connections**: MySQL, PostgreSQL, SQL Server connections
- **File Uploads**: CSV and Google Sheets file integration
- **API Connections**: Custom API integrations and webhooks

### Visualization Capabilities
- **Chart Types**: Comprehensive library of charts and graphs
- **Interactive Reports**: Dynamic filtering and drill-down capabilities
- **Custom Visualizations**: Community-built custom chart components
- **Responsive Design**: Mobile-optimized report layouts
- **Real-Time Data**: Live data connections and automatic refresh

### Collaboration Features
- **Sharing**: Granular sharing permissions and access control
- **Comments**: Collaborative commenting on reports and data points
- **Version History**: Track changes and revert to previous versions
- **Embedding**: Embed reports in websites and applications
- **Export Options**: PDF, image, and data export capabilities

## Data Management

- **Data Freshness**: Control data refresh frequency and caching
