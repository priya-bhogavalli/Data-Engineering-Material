# Sisense - Key Concepts

## Overview
Sisense is an AI-driven analytics platform that simplifies complex data analysis by providing intuitive tools for data preparation, visualization, and embedded analytics. It's designed to handle large datasets and make analytics accessible to business users.

## Core Architecture

### Single-Stack Architecture
- **Unified Platform**: Complete analytics stack in single solution
- **In-Chip Technology**: Optimized for modern CPU architectures
- **Columnar Database**: High-performance columnar storage engine
- **Distributed Processing**: Scale-out architecture for large datasets

### ElastiCube Technology
- **Data Engine**: Proprietary in-memory analytics engine
- **Compression**: Advanced compression algorithms
- **Parallel Processing**: Multi-threaded query execution
- **Automatic Optimization**: Self-optimizing performance
