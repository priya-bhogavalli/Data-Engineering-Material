# HTML - Key Concepts

## Overview
HTML (HyperText Markup Language) is the standard markup language for creating web pages, providing structure and semantic meaning to web content.

## Document Structure

### Basic Structure
- **DOCTYPE**: Document type declaration
- **html**: Root element
- **head**: Metadata container
- **body**: Visible content container
- **title**: Page title

### Semantic Elements
- **header**: Page/section header
- **nav**: Navigation links
- **main**: Primary content
- **article**: Standalone content
- **section**: Thematic grouping
- **aside**: Sidebar content
- **footer**: Page/section footer

## Content Elements

### Text Elements
- **Headings**: h1-h6 hierarchy
- **Paragraphs**: p elements
- **Lists**: ul, ol, li
- **Emphasis**: em, strong
- **Line breaks**: br, hr

### Media Elements
- **Images**: img with alt text
- **Audio**: audio element
- **Video**: video element
- **Canvas**: Drawing surface
- **SVG**: Scalable vector graphics

### Interactive Elements
- **Links**: a elements with href
- **Buttons**: button elements
- **Forms**: form, input, textarea
- **Details**: Collapsible content
- **Dialog**: Modal dialogs

## Forms & Input

### Form Elements
- **form**: Form container
- **input**: Various input types
- **textarea**: Multi-line text
- **select**: Dropdown menus
- **button**: Form buttons

### Input Types
- **text**: Single-line text
- **email**: Email validation
- **password**: Hidden text
- **number**: Numeric input
- **date**: Date picker
- **file**: File upload
- **checkbox**: Boolean selection
- **radio**: Single selection

### Validation
- **required**: Mandatory fields
- **pattern**: Regex validation
- **min/max**: Value constraints
- **maxlength**: Character limits
- **Custom validation**: JavaScript validation

## Attributes

### Global Attributes
- **id**: Unique identifier
- **class**: CSS class names
- **style**: Inline styles
- **data-***: Custom data attributes
- **title**: Tooltip text

### Accessibility Attributes
- **alt**: Image descriptions
- **aria-***: ARIA attributes
- **role**: Element roles
- **tabindex**: Tab order
- **lang**: Language specification

## Best Practices
- **Semantic HTML**: Use appropriate elements
- **Accessibility**: Include ARIA and alt text
- **Validation**: Use proper HTML5 validation
- **Performance**: Optimize images and resources
- **SEO**: Structure for search engines
