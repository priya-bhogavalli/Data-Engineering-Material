# CSS - Key Concepts

## Overview
CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation and styling of HTML documents, controlling layout, colors, fonts, and visual appearance.

## Core Concepts

### Selectors
- **Element**: `p`, `div`, `h1`
- **Class**: `.classname`
- **ID**: `#idname`
- **Attribute**: `[attribute="value"]`
- **Pseudo-classes**: `:hover`, `:focus`, `:nth-child()`

### Box Model
- **Content**: Inner content area
- **Padding**: Space inside border
- **Border**: Element boundary
- **Margin**: Space outside border
- **Box-sizing**: Content-box vs border-box

### Layout Systems
- **Flexbox**: One-dimensional layout
- **Grid**: Two-dimensional layout
- **Float**: Legacy positioning
- **Position**: Static, relative, absolute, fixed, sticky
- **Display**: Block, inline, inline-block, none

## Styling Properties

### Typography
- **Font-family**: Typeface selection
- **Font-size**: Text size
- **Font-weight**: Text thickness
- **Line-height**: Vertical spacing
- **Text-align**: Horizontal alignment

### Colors & Backgrounds
- **Color**: Text color
- **Background-color**: Element background
- **Background-image**: Image backgrounds
- **Gradients**: Linear and radial gradients
- **Opacity**: Transparency levels

### Spacing & Sizing
- **Width/Height**: Element dimensions
- **Margin**: External spacing
- **Padding**: Internal spacing
- **Min/Max**: Size constraints
- **Units**: px, em, rem, %, vh, vw

## Advanced Features

### Responsive Design
- **Media Queries**: Device-specific styles
- **Viewport Units**: vw, vh, vmin, vmax
- **Flexible Units**: em, rem, %
- **Mobile-first**: Progressive enhancement
- **Breakpoints**: Screen size thresholds

### Animations & Transitions
- **Transitions**: Property change animations
- **Keyframes**: Complex animations
- **Transform**: 2D/3D transformations
- **Animation Properties**: Duration, timing, delay
- **Performance**: GPU acceleration

### Modern CSS
- **Custom Properties**: CSS variables
- **Calc()**: Mathematical calculations
- **Grid Areas**: Named grid regions
- **Flexbox Gap**: Spacing between items
- **Container Queries**: Element-based queries

## Best Practices
- **Semantic HTML**: Meaningful structure
- **Mobile-first**: Responsive design approach
- **Performance**: Optimize loading and rendering
- **Maintainability**: Organized, readable code
- **Accessibility**: Inclusive design principles
