# Alteryx - Key Concepts

## Overview
Alteryx is a self-service data analytics platform that enables users to prepare, blend, and analyze data through a visual, drag-and-drop interface.

## Core Components
- **Alteryx Designer**: Visual workflow development environment
- **Alteryx Server**: Enterprise deployment and collaboration platform
- **Alteryx Connect**: Data catalog and collaboration hub
- **Alteryx Promote**: Model deployment and management
- **Alteryx Intelligence Suite**: Advanced analytics and ML capabilities

## Key Features
- **Visual workflow**: Drag-and-drop interface for data workflows
- **Data preparation**: Comprehensive data cleansing and transformation
- **Advanced analytics**: Statistical analysis and predictive modeling
- **Spatial analytics**: Geographic and location-based analysis
- **Machine learning**: AutoML and custom ML model development
- **Data blending**: Combine data from multiple sources

## Data Connectivity
- **Databases**: SQL Server, Oracle, MySQL, PostgreSQL, Teradata
- **Cloud platforms**: AWS, Azure, Google Cloud, Snowflake
- **Files**: Excel, CSV, JSON, XML, Parquet, SAS, SPSS
- **APIs**: REST APIs, web services, OData
- **Applications**: Salesforce, SAP, Marketo, Google Analytics

## Workflow Tools
- **Input/Output**: Data input and output tools
- **Preparation**: Data cleansing, parsing, and transformation
- **Join**: Various join operations and data blending
- **Transform**: Aggregation, filtering, and data manipulation
- **Predictive**: Machine learning and statistical analysis
- **Spatial**: Geographic analysis and mapping tools
